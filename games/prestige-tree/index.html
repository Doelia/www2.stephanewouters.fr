Prestige Tree: Mobile

Mobile version of the incremental game "Prestige Tree," originally created by Jacorb90.

Progress through a layer tree, accumulate points to unlock new upgrades, milestones and access to complexifed layers. Make strategic decisions to advance more quickly and reach the ultimate nodes of the tree.

Transfer your save from the desktop version to the mobile version and back again, enabling seamless gameplay wherever you are.
